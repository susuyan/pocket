# OneV 风格技术文章大纲模板 v1.0

1. 背景与问题
- 背景与场景
- 核心认知问题
- 成功判据

2. 直觉解法
- 天真方案
- 直觉的吸引力
- 在何种前提下有效

3. 问题显现
- 失败点与反例
- 隐性假设被打破的位置
- 规模化/长期维护下的问题

4. 约束与目标
- 环境与资源约束
- 非功能目标（成本、稳定性、速度等）
- 权衡因素

5. 实际方案
- 选择与核心理由
- 与备选的对比与取舍
- 适用范围

6. 实践细节
- 关键实现点
- 落地要诀
- 踩坑与边界

7. 局限性
- 不适用场景
- 已知缺陷
- 未来演进方向

8. 总结
- 关键决策复盘
- 面向读者的启示
- 后续可拓展方向
